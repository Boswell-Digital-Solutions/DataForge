# ЁЯза Ecosystem Role

DataForge acts as the **shared memory layer** for the Forge Suite:

- **AuthorForge** тАУ writing knowledge, genre structures, pacing, scene-level analysis  
- **NeuroForge** тАУ model routing, embeddings, context retrieval  
- **VibeForge** тАУ execution context, prompt analytics, evaluation data  
- **TradeForge** тАУ market signals, historical feeds, structured datasets  
- **Leopold** тАУ ecological observations, biological datasets  
- **Livy** тАУ historical data, geospatial narratives  

Every Forge product consumes DataForge as the **source of truth**, ensuring consistency, compliance, and cross-product intelligence.
